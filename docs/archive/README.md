# Archive

Здесь хранятся завершенные задачи и старые планы, которые больше не должны смешиваться с активным backlog.

## Rules

- Переносить task-файл сюда только после выполнения задачи.
- Перед переносом заполнить секцию `Result`.
- Использовать имя файла в формате `YYYY-MM-DD-short-task-name.md`.
- В `docs/roadmap.md` оставлять короткую ссылку на архивированную задачу, если она важна для истории проекта.
